搭建个人博客 之 Hexo基础

一直有在本地写md笔记的习惯，又想同步到自己的各个设备随时查看，Win平台试过马克飞象同步到印象笔记，印象笔记端不能编辑，放弃……之后一直使用typora在本地保存，自从18年换到MacOS平台后，尝试过MWeb，Quiver，Bear，Ulysses，最后选择MWeb。后来有了自己的服务器，使用了WordPress一段时间，感觉比较麻烦，速度也不理想。折腾向的我又选择了Hexo，所以有了这篇文章，哈哈。现在是采用MWeb编辑md文件，通过MWeb发布服务直接推到服务器上，图片通过MWeb上传到七牛云，服务器使用Docker和Nginx来实现。这样一来，写完md文件后，直接发布，相当方便哈哈！

安装Hexo

Mac已经使用HomeBrew安装了node和git，所以直接安装Hexo。

新建并进入Hexo目录，并执行Hexo初始化。




可以在http://localhost:4000看到本地的hexo博客。

使用Hexo写作

layout有三种布局：

  • post 文章
  • page 页面
  • draft 草稿

三种布局的模版在Hexo目录/scaffolds/下，可以根据需要进行修改，正文部分可以使用md格式也可以使用html格式。

配置Hexo站点

在Hexo目录下的_config.yml是Hexo的站点配置文件，注意每个冒号后要有空格，这是我做更改的配置，其余都为默认。

如果只部署到服务器上(更多方式请参考 : hexo.io) ：

使用Hexo主题

主题可以在hexo.themes上选择并下载，默认主题是landscape，我下载了Modernist和NexT。

安装NexT主题

进入Hexo目录。

修改Hexo路径下站点配置文件_config.yml。

配置NexT主题

在Hexo目录下cd themes/next可以看到_config.yml，这个是NexT主题的配置文件，注意不要跟Hexo的站点配置文件搞混。

菜单配置

设置后需要在Hexo主目录下创建页面：

如果引入评论系统要在页面加上comments: false（分类和关于页面也要加）

更改主题布局

四种布局可选，打开所需布局即可。

联系方式

头像

url可以是OSS外链，也可以在主题下的source/images/下放置头像文件即可。

边栏

自带动画效果

开启动画会影响打开网站速度，可以关闭

自带动画背景效果

NexT主题自带动画canvas_nest或three_waves，根据需求设置值为true或者false即可。

fancybox图片放大

在NexT主题配置文件中设置fancybox: true即可。

自定义行内代码样式

进入themes/next/source/css/_custom修改custom.styl文件即可。

百度统计分析

  1. 注册百度统计，并新建网站列表。
  2. 建立成功后，获取新版统计代码。

  3. 将上图中黑色遮挡的ID复制到NexT配置文件,即可开启。

百度统计的自动代码检测可能检测不到代码装载，可以进行手动检查。

不蒜子阅读量统计

在NexT配置文件开启即可。

评论系统Valine

NexT已经集成了Valine，在主题的配置文件中就可以找到。Valine是基于LeanCloud的，所以需要先申请LeanCloud。

  1. 在LeanCloud创建应用。
  2. 获取AppId和AppKey
  3. 在NexT主题配置文件找到valine并配置

使用Valine阅读量统计需要注意：leancloud-counter-security is not supported for now. When visitor is set to be true, appid and appkey are recommended to be the same as leancloud_visitors’ for counter compatibility. Article reading statistic
